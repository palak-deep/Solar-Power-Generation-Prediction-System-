# Solar-Power-Generation-Prediction-System-
A full-stack machine learning project that forecasts solar panel energy output for any location worldwide. Trained on 8,300+ hours of historical solar generation, irradiance, and weather data (2020–2021), the XGBoost model achieves R² = 0.88 with an MAE of ~1 kWh. The live app fetches real-time 7-day weather forecasts .
